# Tiroid Papiller Karsinom Rapor Oluşturucu - Statik React

Bu paket GitHub Pages icin build gerektirmeyen statik React surumudur.

## Kurulum

1. Zip dosyasini acin.
2. `index.html`, `.nojekyll` ve `README.md` dosyalarini GitHub repo kok dizinine kopyalayin.
3. GitHub Pages kaynagini `main` branch / root olarak ayarlayin.
4. Sayfa: `https://metinciris.github.io/Tiroid-papiller-raporlama/`

## Ozellikler

- React ile tek sayfa statik uygulama.
- Yazilan veya isaretlenen alan kisa sure parlar.
- Dolu alanlar turuncu tonlu arka plan alir.
- Kotu prognostik bulgular kirmizi vurgulanir.
- Aktif tumor odagi kart ve ust chip olarak belirginlesir.
- Coklu tumor odaklarinda rapor daha okunakli bloklar halinde uretilir.
- Rapor onizleme kaydirilabilir ve tek tikla kopyalanabilir.
- Veriler tarayici localStorage alaninda saklanir.

Not: Bu surum React, ReactDOM ve Babel dosyalarini CDN uzerinden yukler. GitHub Pages'te dogrudan calisir; npm install veya build gerektirmez.
